Get started in minutes

Three simple steps to beautiful documentation

1

Add the gem

Add RailsDocGenerator to your Gemfile and run bundle install.

# Gemfile
gem 'rails_doc_generator'
2

Run the generator

Run the install generator to create the initializer and mount the engine.

$ rails generate rails_doc_generator:install
3

Visit your docs

Start your server and visit the documentation URL. That's it!

$ rails server
# Visit http://localhost:3000/api-doc
